# Inkling Frontend

React frontend application for the Inkling learning platform.

## Setup

1. Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm start
```

The app will open at http://localhost:3000

## Build

Build for production:
```bash
npm run build
```

## Backend API

The frontend expects the backend API to be running on http://localhost:8000. 
Make sure to start the backend server first (see main project README).
